1. Introduction

Welcome! This data science project analyzes lifestyle and medical factors like age, BMI, and sleep duration. It predicts the presence and type of sleep disorders such as Insomnia or Sleep Apnea. Sleep disorders can seriously impact health and well-being. Early detection enables timely intervention to improve sleep quality and overall health.

The applied machine learning models are:

  • Decision Tree Classifier
  • Random Forest Classifier

To showcase the results, a web application was developed using Flask and deployed within a Docker container. All project files and detailed documentation are available on my GitHub profile.

2. Problem Statement

Understanding job market trends for Data Scientist roles is crucial for both job seekers and employers.

This project aims to address the following key questions:

  • Identifying Key Influencers: What factors (such as, location, required skills, company size, and industry etc.) influence salary estimates for Data Scientist roles?
  • Analyzing Salary Variations: How do company size, location, and industry correlate with job salary variations?
  • Most Demanding Skill: Which skills and qualifications are most frequently listed in high-paying job postings?
  • Predictive Modeling: Can we build a predictive model to estimate salaries based on job features?
  • Interactive Visualization: Visualing the data in an interactive manner using Power BI Dashboard.

7. Model Building

The following machine learning models were used:

  • Model 1: Linear Regression
  • Model 2: Lasso Regression
  • Model 3: Random Forest

The best-performing model was Random Forest with an accuracy of 84.89%.

8. Conclusion

This project provides a comprehensive analysis of data scientist job listings, offering valuable insights for both job seekers and employers. By leveraging data scraped from Glassdoor, the project identifies key factors influencing job offerings and salaries, analyzes job trends, and explores correlations between various job attributes.

The use of predictive modeling, particularly the Random Forest model, allows for accurate salary estimations based on job details. The interactive web application developed using Flask and deployed in a Docker container enables users to visualize and explore these insights effectively.
